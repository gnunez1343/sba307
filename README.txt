# Products Page Project

## **Description**
This project is a simple, responsive website that showcases a product catalog with information and images for each product. It was built using HTML5, CSS3, and Bootstrap 5.

The site includes the following:
- **Header:** Displays the website title.
- **Navbar:** Links to Admin, User, and Products pages.
- **Main Section:** Displays three product cards with images, descriptions, and a "Buy Now" button.
- **Footer:** Includes copyright information.

## **Technical Specifications**
- **Languages/Frameworks:** HTML5, CSS3, Bootstrap 5.
- **Responsiveness:** Fully responsive layout compatible with desktop and mobile devices.
- **External Libraries:** Bootstrap 5.3 (via CDN).
- **Assets:**
  - Product images sourced from royalty-free platforms (Pixabay, Pexels).
  - External CSS file for custom styling.

## **How to Run the Project**
1. Download the project folder.
2. Open `index.html` in a browser to view the homepage.
3. Navigate through links in the navbar.
